Modern Mobile phones and hand held computers have enough processing power, screen resolution and size to run popular games such as first person shooter games, role playing games and the similar. However, these games, when run on a computer, usually require both keyboard to control movement and arms, and mouse for aiming and shooting. On most modern mobile computing devices, control is only available through touch screen and accelerometer gestures (movement of the device), which do not offer the same granular control; Touching the screen hides part of it, and the 'fat finger syndrome' makes it hard for fine movement or aiming controls. It is possible to hook up a mouse and keyboard to these modern mobile computing devices through the Bluetooth interface but this renders it non-mobile. A wearable controller operated by one-hand, that operate well while in movement, functions in any light conditions and provides enough game and application controls for mobile computing devices is therefore needed.

ΦR is a one-hand controller shaped as thumb ring that provides rich control suitable for gaming. ΦR detects hand and fingers movements and controls the mobile platform via Bluetooth. The ring has two main detectors; a camera located on the inner part of the ring and detects finger movements and a mouse control detector that detects hand movements.
